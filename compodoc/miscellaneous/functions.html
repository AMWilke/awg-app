src/testing/detect-changes-on-push-helper.ts

detectChangesOnPush
detectChangesOnPush(cf)

Test helper method: detectChangesOnPush.

It provides a workaround to use fixture.detectChanges with ChangeDetection.onPush strategy Cf. https://github.com/angular/angular/issues/12313#issuecomment-528536934

Has to be used with await keyword inside async function:

Parameters :
Name Optional
cf No
Example : 
it(&#39;... should do something&#39;, async () =&gt; {
    component.inputVariable = changedValue
    await detectChangesOnPush(fixture);
});
